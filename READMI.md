# Letter Vocabulary Master

A fun terminal-based English vocabulary guessing game with Persian meanings, IPA pronunciation, and example sentences.
Guess letters one by one to reveal English words (with Persian meanings)

Educational console game for learning English vocabulary through letter-by-letter guessing.
Features three difficulty levels, scoring system, high-score table, Persian translations, IPA pronunciation, and example sentences.

## Main Features

- 3 difficulty levels (Easy / Medium / Hard)
- 30+ vocabulary words with Persian meaning + IPA + example sentence
- Points system + bonus for remaining attempts
- Saves top 10 high scores
- Colorful terminal animations & feedback
- Full instructions + end-game statistics

## Requirements

- Python 3.8 or higher
- colorama library

## Installation & Running

```bash
# 1. Clone the repository
git clone https://github.com/YOUR-USERNAME/letter-vocabulary-master.git
cd letter-vocabulary-master

# 2. Install dependencies
pip install -r requirements.txt

# 3. Run the game
python letter_game.py
# or
python3 letter_game.py
